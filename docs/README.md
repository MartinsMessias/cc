## Como navegar

- Se você quer entender **como a CLI inicia**, comece por `02-fluxo-execucao.md`.
- Se quer entender **o que cada diretório faz**, use `03-arquitetura-pastas.md`.
- Se quer mapear **funcionalidades ao código**, combine `04` + `05` + `07`.
- Se quer uma visão “catálogo” de alto nível, abra `08-inventario.md`.


- Se você quer **mapear estratégia de execução por cenário**, leia `09-estrategias-por-caso.md`.

- Se você quer **replicar o projeto a partir da documentação**, siga `10-guia-replicacao.md`.
